The API behind every
Scribe language pack.

Scribe-Server is the backend service that makes Scribe's language data available for download. Built on regularly updating Scribe-Data datasets, it signals when new data is ready and serves it as versioned language packs to every Scribe app.

The data pipeline

Where Scribe-Server fits

Scribe-Server is one link in a small, transparent pipeline — each stage has its own repository and its own maintainers.

01
Wikidata & Wiktionary
Community-sourced grammar, translations and lexemes, freely licensed.
02
Scribe-Data
Queries and formats raw Wikidata into structured JSON contracts or SQlite files.
03
Scribe-Server
Serves contracts over a REST API and bundles them into versioned SQLite packs.
you are here
