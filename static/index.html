Scribe-Server

Backend API for Wikidata-powered language data

Scribe-Server is a backend service that provides the API by which data is available for download within Scribe applications. The goal is to create a Scribe-Data based regularly updating dataset that can signal new data availability as well as allow for language pack downloads. The Scribe-Server API can be used as a data source for pre-processed language data.

API Endpoints

  • GET /api/v1/languages - List available languages
  • GET /api/v1/contracts[?lang_iso=xx] - Get contracts (optional language filter)
  • GET /api/v1/data/:lang_iso - Get full language data with schema
  • GET /api/v1/data-version/:lang_iso - Get version info for a language
  • GET /api/v1/language-stats?codes=fr,de - Get statistics for all or selected languages
